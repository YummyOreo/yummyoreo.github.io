What is TBFighters?

TBFighters is a group of public health activists who fight the causes of tuberculosis.

Stack

The website is pure HTML, CSS, and Javascript. We tend to stray away from using Javascript as much as possible and avoid big animations or page movements. This is an attempt to make the website not only simple to make, but also accessible to anyone that wants to use it. We also prioritize web accessible at every turn, aiming for our website to be able to be used by anyone, regardless of disability. The website is hosted on Cloudflare pages and there is a separate redirect server that is also hosted on Cloudflare pages.
